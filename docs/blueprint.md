# **App Name**: eBoutique Lite

## Core Features:

- Product Listing: Display products with their images and descriptions.
- Search Bar: A search bar allowing users to search for products by name.
- Shopping Cart Indicator: Display a number next to the shopping cart icon to show the amount of items in it
- Price Filtering: Filter products by price range.
- Type Filtering: Filter products by type (e.g., electronics, appliances, cellphones).
- Product Image: Have a functional display image for each of the products listed on the online store.

## Style Guidelines:

- Primary color: Blue (#007BFF) to convey trust and reliability.
- Background color: White (#FFFFFF) for a clean, modern look.
- Accent color: Black (#000000) to draw attention to key interactive elements such as 'Add to Cart'.
- Body and headline font: 'Inter', a sans-serif, providing a modern, neutral feel for both headlines and body text.
- Use minimalist icons for navigation and filters.
- Clean and responsive design optimized for various screen sizes, ensuring products are clearly visible.
- Subtle transitions and hover effects for product interactions.
